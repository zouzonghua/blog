我的 VSCode 插件配置

发表于 2019-08-18 更新于 2023-03-01 分类于 软件

导读

工欲善其事必先利其器!

工匠想要使他的工作做好，一定要先让工具锋利。

软件开发者每天接触的就是编辑器，只有把编辑器熟练的掌握才能更好的提升开发效率。

拓展插件(Extensions)

  1. ESLint - 插件化的 javascript 代码检测工具，有多种主流的编码风格规则集可供选择,典型的有 Airbnb、Standard 等代码规范
  2. TSLint - Typescript 的语法高亮，语法提示。
  3. stylelint - css 代码检查工具。（标准化 stylelint 的插件，规范 style 代码）
  4. markdownlint - Markdown 编写文档规范检查工具，没有规范是不行的。
  5. Code Spell Checker - 这款插件能实时的识别单词拼写是否有误，并给出提示，不少 bug 都是因为拼写错误导致的。
  6. Bracket Pair Colorizer - 增加代码中括号之间嵌套关系的颜色区分。
  7. Trailing Spaces - 检查代码中多出空格，强迫症必备。
  8. Vetur - 开发 Vue 必装的一个插件。
  9. ES7 React/Redux/GraphQL/React-Native snippets - ES7 React/Redux/GraphQL/React-Native snippets 常用代码片段。
  10. koroFileHeader - 在 vscode 中用于生成文件头部注释和函数注释的插件。
  11. GitLens — Git supercharged - 一款强大 GIt 源代码管理插件。
  12. Live Server - 快速启动一个本地服务预览项目。
  13. open in browser - 在浏览器中快速打开 html 文件。
  14. Path Intellisense - VScode 中路径智能补全。
  15. npm Intellisense -自动引入 node_modules 中的 JS 路径。
  16. Project Manager - 项目管理插件。
  17. Settings Sync - 绑定 GitHub 账户进行插件配置备份恢复。
  18. vscode-icons - 为侧边栏文件类型提供相应的图标。
  19. Oceanic Next - React 官方文档推荐的一款主题。
  20. Sublime Babel - React 官方文档推荐的一款 ES6+ 语法高亮。

VSCode Settings Sync 插件的使用

发表于 2019-07-26 更新于 2023-03-01 分类于 插件

导读

Settings Sync 是 VSCode 编辑器上的一款插件，可以将当前编辑器所有配置和插件同步到 GitHub 上，以后只需要从 GitHub 上获取，就可以一次性恢复插件和配置。

适用于

  • Windows 系统
  • MacOS 系统
  • Ubuntu 系统

前序准备

  • 下载安装 VSCode 编辑器
  • 注册申请 GitHub 帐号

快速开始

第一步

  • 先在 VSCode 编辑器里面搜索安装 Settings Sync 插件 ，安装好之后重启编辑器加载插件。

第二步

  • 按下 Shift + Alt + U （备份）

    • Upload Key : Shift + Alt + U \ Shift ⇧+Option ⌥+ U
    • Download Key : Shift + Alt + D \ Shift ⇧+Option ⌥+ U

  • 弹出 Settings Sync 插件界面选择 Login with Github 绑定 GItHub 帐号

第三步

  • 弹出浏览器界面 GitHub 登录界面，输入帐号密码进行下一步确定

第四步

  • 确定授权之后页面显示成功就可以关闭当前 tab 页面了

第五步

  • 回到 VSCode 编辑器会多出一个 tab 窗口点击 Skip 创建新 Gist（以前旧版插件需要到 GitHub 上面手动创建 Gist，现在更新了之后非常方便）

  • 按下 Shift + Alt + U 进行当前插件和配置的备份，重装电脑之后或者更换系统之后只需重新下载 VSCode 和 Settings Sync 插件进行以上步骤绑定（如果是已有备份记录的用户只需要选中对应的 Gist ），按下 Shift + Alt + D 进行就可以完成 插件和配置的备份 恢复了。

我的 Mac 软件清单

发表于 2019-07-26 更新于 2023-03-01 分类于 软件

导读

工欲善其事必先利其器!

工匠想要使他的工作做好，一定要先让工具锋利。

好的软件在于精不在于多

操作系统

  • macOS High Sierra（10.13.6） - macOS High Sierra是苹果公司Mac计算机系列产品的操作系统

开发工具

  • Homebrew - macOS 缺失的软件包的管理器
  • iTerm2 - MAC 下最好的终端工具 brew cask install iterm2
  • typora - 一款轻便简洁的Markdown编辑器 brew cask install typora
  • Yarn - 快速、可靠、安全的依赖管理 brew install yarn --without-node
  • Node.js - 是一个基于Chrome V8 引擎的JavaScript 运行环境 brew install node.js
  • Visual Studio Code - 微软推出的免费/开源编辑器，前端开发必备 brew cask install visual-studio-code

版本控制

  • Xversion - Xversion Mac版是一款macOS平台的Subversion版本控制工具。
  • Git - Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目 brew install git

浏览器

  • Google Chrome - Google Chrome是由Google开发的免费网页浏览器 brew cask install google-chrome

IM 工具

  • 微信 - 微信，是一个生活方式 brew cask install wechat
  • QQ - I’m QQ - 每一天，乐于沟通 brew cask install qq

待办事项工具

  • Microsoft To-Do - 从工作到休闲，To-Do 都是日常完成任务的最简便方式

系统相关工具

  • Dozre - 隐藏 macOS 菜单栏的图标
  • Tooth Fairy - 可以帮你一键切换 AirPods
  • CleanMyMac - 让 Mac 更流畅好用的垃圾清理系统优化应用
  • Tuxera NTFS - Mac 上的 NTFS 文件系统驱动 macOS High Sierra(10.13.6)
  • Fliqlo - 翻页时间屏保 brew cask install fliqlo
  • Mos - 让你的鼠标滚轮丝滑如触控板 brew cask install mos
  • Snipaste - 一个简单但强大的截图工具 brew cask install snipaste
  • Alfred - Alfred可以说是Mac上的一款效率神器 brew cask install alfred
  • The Unarchiver - 解压许多不同种类的归档压缩文件 brew cask install the-unarchiver
  • ShiftIt - ShiftIt 是 Mac 平台上的一款窗口位置和大小管理软件 brew cask install shiftit
  • Karabiner Elements - Karabiner-Elements是macOS Sierra（10.12）或更高版本上用于键盘自定义的强大工具 brew cask install karabiner-elements

远程协助

  • TeamViewer - 远程协助及在线协作和会议功能的软件，商业软件个人使用免费 brew cask install teamviewer

影音播放

  • IINA - 一个现代的 macOS 视频播放器 brew cask install iina
  • 网易云音乐 - 网易云音乐是一款专注于发现与分享的音乐产品，依托专业音乐人、DJ、好友推荐及社交功能，为用户打造全新的音乐生活。 brew cask install neteasemusic

阅读学习

  • NeatReader - 新世代ePub 阅读器，创造美好阅读时光
  • 欧路词典 - 权威的英语词典软件，英语学习者必备的工具 brew cask intsall eudic
