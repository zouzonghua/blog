我的 VSCode 插件配置

发表于 2019-08-18 更新于 2023-03-01 分类于 软件

导读

工欲善其事必先利其器!

工匠想要使他的工作做好，一定要先让工具锋利。

软件开发者每天接触的就是编辑器，只有把编辑器熟练的掌握才能更好的提升开发效率。

拓展插件(Extensions)

  1. ESLint - 插件化的 javascript 代码检测工具，有多种主流的编码风格规则集可供选择,典型的有 Airbnb、Standard 等代码规范
  2. TSLint - Typescript 的语法高亮，语法提示。
  3. stylelint - css 代码检查工具。（标准化 stylelint 的插件，规范 style 代码）
  4. markdownlint - Markdown 编写文档规范检查工具，没有规范是不行的。
  5. Code Spell Checker - 这款插件能实时的识别单词拼写是否有误，并给出提示，不少 bug 都是因为拼写错误导致的。
  6. Bracket Pair Colorizer - 增加代码中括号之间嵌套关系的颜色区分。
  7. Trailing Spaces - 检查代码中多出空格，强迫症必备。
  8. Vetur - 开发 Vue 必装的一个插件。
  9. ES7 React/Redux/GraphQL/React-Native snippets - ES7 React/Redux/GraphQL/React-Native snippets 常用代码片段。
  10. koroFileHeader - 在 vscode 中用于生成文件头部注释和函数注释的插件。
  11. GitLens — Git supercharged - 一款强大 GIt 源代码管理插件。
  12. Live Server - 快速启动一个本地服务预览项目。
  13. open in browser - 在浏览器中快速打开 html 文件。
  14. Path Intellisense - VScode 中路径智能补全。
  15. npm Intellisense -自动引入 node_modules 中的 JS 路径。
  16. Project Manager - 项目管理插件。
  17. Settings Sync - 绑定 GitHub 账户进行插件配置备份恢复。
  18. vscode-icons - 为侧边栏文件类型提供相应的图标。
  19. Oceanic Next - React 官方文档推荐的一款主题。
  20. Sublime Babel - React 官方文档推荐的一款 ES6+ 语法高亮。
